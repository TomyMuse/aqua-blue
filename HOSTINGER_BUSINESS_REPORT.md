# Informe Detallado: Características de Hostinger Business

Este informe detalla las especificaciones técnicas, capacidades de desarrollo y servicios incluidos en el plan **Hostinger Business**, una solución diseñada para proporcionar herramientas avanzadas y potencia para el crecimiento de aplicaciones y sitios web en un entorno completamente gestionado.

## 1. Especificaciones de Hardware y Rendimiento
El plan Business ofrece una infraestructura optimizada para manejar proyectos dinámicos con un rendimiento superior al hosting compartido básico:
- **Procesamiento:** Incluye 2 núcleos de CPU.
- **Memoria:** Dispone de 3 GB de RAM.
- **Almacenamiento:** Cuenta con 50 GB de almacenamiento NVMe, descrito como uno de los más rápidos del mundo.
- **Ancho de banda:** Proporciona tráfico ilimitado, permitiendo que la aplicación escale libremente con un precio mensual fijo.
- **CDN Global:** Incluye de forma gratuita una Red de Entrega de Contenidos (CDN) propia de Hostinger para mejorar la velocidad de carga a nivel mundial.

## 2. Capacidades de Alojamiento y Aplicaciones
Este plan permite centralizar múltiples proyectos bajo una sola suscripción:
- **Sitios Web:** Soporta hasta 50 sitios web.
- **Aplicaciones Node.js:** Permite gestionar hasta 5 aplicaciones web de Node.js de forma simplificada.
- **Compatibilidad de Tecnologías:** Es compatible con proyectos construidos en Node.js, PHP/HTML, WordPress y el Creador de Sitios de Hostinger.

## 3. Herramientas de Desarrollo y Despliegue Agéntico
El plan Business destaca por su integración con flujos de trabajo modernos de *vibe coding* y automatización:
- **Agentic Deployment (Nuevo):** Permite desplegar aplicaciones directamente desde el IDE (como Cursor, VS Code o Claude) sin necesidad de acceder al panel de control web.
- **Integración con GitHub:** Incluye despliegues automáticos cada vez que se realiza un push al repositorio conectado.
- **Detección Automática de Frameworks:** El sistema detecta y gestiona automáticamente los comandos de construcción para frameworks populares como React, Next.js, Vue.js, Express.js, Angular, y Vite, entre otros.
- **Gestión de Variables de Env:** Permite configurar claves de API y credenciales de bases de datos de forma segura desde el panel, con opción de importar archivos `.env`.

## 4. Funciones de Inteligencia Artificial (IA)
Hostinger integra herramientas de IA para facilitar la creación y optimización de sitios:
- **Web2Agent (Experimental):** Transforma el sitio web en un agente totalmente compatible con el Protocolo de Contexto de Modelo (MCP), permitiendo que herramientas como Claude Desktop o Cursor comprendan y rastreen el contenido del sitio de forma inteligente.
- **Creadores de Sitios con IA:** Incluye un constructor de sitios web y de e-commerce impulsado por IA para lanzamientos rápidos.
- **Gestión de Tráfico de Bots:** Utiliza IA para gestionar el tráfico de bots y proteger los recursos del servidor.

## 5. Seguridad y Respaldo
La protección de datos está integrada de forma nativa en el plan:
- **Seguridad Activa:** Incluye un Firewall de Aplicaciones Web (WAF) y protección contra ataques DDoS.
- **Certificados SSL:** Proporciona certificados SSL gestionados gratuitos de por vida.
- **Backups:** Ofrece copias de seguridad diarias y la posibilidad de generar backups bajo demanda para asegurar la integridad de la información.

## 6. Servicios de Valor Agregado
Para complementar la presencia online, el plan incluye:
- **Dominio Gratuito:** Registro de un nombre de dominio gratis por un año.
- **Correo Profesional:** Hasta 5 casillas de correo electrónico gratuitas por un año para cada sitio web.
- **Bases de Datos:** Incluye bases de datos MySQL gestionadas, donde Hostinger se encarga de la estabilidad y el mantenimiento.

---
*Este plan se posiciona como una alternativa económica y potente a plataformas como Vercel o Netlify para proyectos de Node.js y sitios generados por IA, ofreciendo costos fijos mensuales y una infraestructura simplificada.*
